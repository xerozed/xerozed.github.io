Airbnb
Python Tableau
I worked with a very detailed Airbnb pricing dataset covering several major European cities. It includes features such as room type, host status, ratings, number of bedrooms and so on... After cleaning and preparing the data in Python, I used Tableau to visualise geographic price patterns, guest satisfaction differences, and host quality indicators to highlight the main market differences across European cities.
Analytical Questions
  • How do average Airbnb prices vary across European countries?
  • Which European cities have the highest number of Superhosts?
  • How does guest satisfaction differ between countries?
  • How does the distance from the city center affect pricing?
General Data Description
  • Scope: The dataset covers Airbnb listings across 10 major European capitals
  • Timeframe: Data is segmented into "Weekdays" and "Weekends" to analyze price differences.
  • Main Attributes:
    • Listing Info: Price (Euro), Room type (Private/Shared/Entire home)
    • Host Info: Superhost status (True/False)
    • Location: Distance from city center (km), Distance to metro (km).
    • Quality Metrics: Guest satisfaction overall (0-100)

Data Visualization
How does the distance from the city center affect pricing?
  1. For columns, I created a new bin field from the Dist column which gave me less noisy columns.
  2. For rows, I created a calculated field called Price Combined using the formula AVG(Price).
  3. To compare the cities: I added the City field to the Color shelf.
  4. As a last step, I filtered the distance bins and restricted the X-axis to the 0-15 km range.
How does guest satisfaction differ between countries?
  1. First I dragged the generated Latitude and Longitude fields to the Rows and Columns shelves to create a geographic map layout.
  2. Since the dataset only contained city names, I duplicated the City field from the Data pane to create a new field called Countries, I assigned the Countries field a Geographic Role of “Country/Region” then manually renamed each city to its country's name.
  3. I placed the Countries field onto the Marks --> Detail shelf and added AVG(Guest Satisfaction Overall) to Color.
  4. Finally, I adjusted the color scale to a diverging yellow-to-green palette which made the geographic differences much clearer, highlighting which countries tend to receive higher guest satisfaction scores compared to others.
How do average Airbnb prices vary across European cities?
  1. For rows: I dragged the City field onto Rows to list all cities included in the dataset.
  2. For columns: I placed AGG(Price Combined) onto Columns to calculate each city's average Airbnb price.
  3. To enhance visual comparison, I dragged AGG(Price Combined) onto the Color shelf and applied a five-step Red-Green Diverging palette.
Which European cities have the highest number of Superhosts?
  1. For this chart, I dragged the City dimension onto the Columns shelf so that each city appears as a separate bar along the horizontal axis.
  2. I dragged 'Host Is Superhost' onto the Rows shelf, then i changed its Field Role to Count, which counted how many listings belong to each city.
  3. I added CNT(Host Is Superhost) to the Label field so the exact number of superhosts appears on top of each bar.
  4. I also dragged CNT(Host Is Superhost) to Color shelf, so the height and the color shading both reflect the number of superhosts.
  5. In the end, I chose a green gradient palette and applied it, making higher values appear darker green.
